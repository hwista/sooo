## 사용 원칙

- 신규 문서는 아카이브에 추가하지 않습니다.
- 완료된 작업의 기록만 이곳으로 이동합니다.
- 인덱스(`docs/common/README.md`)에는 요약 링크만 유지합니다.

---

## 네이밍 규칙

- 폴더: `{topic}-YYYYMMDD` 형식 (예: `refactoring-20260120`)
- 백업/레거시 파일: 기존 이름에 `-legacy` 접미사 유지
- 대량 이관 작업은 날짜 기준 폴더로 묶습니다.
